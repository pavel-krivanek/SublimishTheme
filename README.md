# SublimishTheme
A dark theme for Pharo. If you like Sublime then you'll like Sublimish theme.

## Install
```smalltalk
Metacello new
    githubUser: 'sebastianconcept' 
    project: 'SublimishTheme' 
    commitish: 'master' 
    path: 'src';
    baseline: 'SublimishTheme';
    onWarningLog;
    load```
